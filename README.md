# Multiagent Online Dating Simulator

This is a resonably extensible simulation framework that I created for my Multiagent Systems course project at Carleton University. It allows running a simulation of a Tinder-like online dating environment. Agents can have different genders, attractiveness levels, premiumness, impostorness, observable/hidden attributes, liking strategies, and compatibility logics (the framework currently has only one compatibility logic). These differences heavily affect the agents' utility (happiness), which depends on their match count and quality (matching with more attractive and more compatible agents yield higher scores in general). It has random and rating-based recommender systems that provide agents some agents to be liked or passed.

Some values, functions, and distributions are modeled after existing studies on dating and romantic relationships. Due to the restrictions of the existing studies and to keep it simpler, the simulator is restricted to two genders and heterosexual agents.
